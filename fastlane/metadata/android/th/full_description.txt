KeePassDX เป็นแอปสำหรับ<b>เก็บและจัดการพาสคีย์และรหัสผ่านต่างๆ</b>ให้คุณได้<b>แก้ไขข้อมูลที่เข้ารหัสในไฟล์เดียว</b>ในรูปแบบไฟล์ KeePass แบบเปิด และ<b>กรอกข้อมูลในฟอร์มต่างๆอย่างปลอดภัย</b>, โดย <b>ไม่จำเป็นต้องเชื่อมต่ออินเตอร์เน็ต</b> และทำงานได้กับมาตรฐานระบบของ แอปนี้ <b>เป็นโอเพ่นซอร์สและไม่มีโฆษณา</b>.

<b>ฟีเจอร์</b>

- พาส์คีย์สำหรับยืนยันตัวตนและเก็บ private keys ไว้ในเครื่อง
- การจดจำด้วยไบโอเมตริกสำหรับการปลดล็อกอย่างรวดเร็ว(ลายนิ้วมือ / ใบหน้า / …)
- จัดการรหัสผ่านแบบใช้ครั้งเดียว (HOTP / TOTP) สำหรับการยืนยันตัวสองปัจจัย (2FA)
- การกรอกข้อมูลอัตโนมัติสำหรับกรอกฟอร์มและรหัสผ่านต่างๆอย่างง่ายดาย
- เมจิคีย์บอร์ดสำหรบกรอกข้อมูลในช่องใดก็ได้
- สร้างไฟล์ฐานข้อมูลแบบเข้ารหัส
- จัดระเบียบบัญชีรหัสผ่านเป็นรายการและหมวดหมู่ต่างๆ
- เปิดและคัดลอกช่องที่อยู่ URI / URL ได้อย่างรวดเร็ว
- เทมเพลตแบบ Dynamic สำหรับรายการต่างๆแต่ละชนิด
- ประวัติการแก้ไขในแต่ละรายการ
- จัดการตั่งค่าต่างๆได้แบบละเอียด
- ดีไซน์ Material พร้อมกับธีม
- รองรับไฟล์ .kdb และ .kdbx files (เวอร์ชั่น 1 ถึง 4) ที่ใช้อัลกอริทึ่ม  AES - Twofish - ChaCha20 - Argon2
- เข้ากันได้กับโปรแกรมทางเลือกส่วนใหญ่ (KeePass, KeePassXC, KeeWeb, …)
- โค้ดของแอปเขียนด้วยภาษาแบบเนทีฟ (Kotlin / Java / JNI / C)

คุณสามารถบริจาคหรือซื้อแบบเวอร์ชั่นโปรเพื่อช่วยสนับสนุนการพัฒนาแอปและฟีเจอร์ต่างๆที่ต้องการได้รวดเร็วขึ้น: <a href="https://play.google.com/store/apps/details?id=com.kunzisoft.keepass.pro">https://play.google.com/store/apps/details?id=com.kunzisoft.keepass.pro</a>
